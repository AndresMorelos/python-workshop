📦 Variables y Tipos de Datos FUNDAMENTOS

Una variable es como una caja con etiqueta donde guardas información. Le das un nombre, y Python recuerda el valor para usarlo después. No necesitas declarar el tipo — Python lo detecta solo.

Ejemplo — Datos de una empresaPython 3
# Texto (str)
nombre_empresa = "Tech Store"
gerente        = "Ana García"

# Número entero (int)
empleados      = 42
año_fundacion  = 2018

# Número decimal (float)
ventas_mes     = 85000.50
tasa_iva       = 0.16

# Verdadero / Falso (bool)
activo         = True
en_quiebra     = False

print(nombre_empresa, "—", empleados, "empleados")
Tipos de datos básicos
int — Entero
Números sin decimales.
empleados = 42
ventas = 1500
float — Decimal
Números con punto decimal.
precio = 99.99
tasa = 0.16
str — Texto
Texto entre comillas.
nombre = "Ana"
ciudad = 'CDMX'
bool — Lógico
Solo True o False.
activo = True
pagado = False
💡 Tip: Los nombres de variables no pueden tener espacios ni caracteres especiales. Usa guión bajo: ventas_mes, nombre_empresa.
📋 Listas COLECCIONES

Una lista guarda varios valores en orden dentro de corchetes [ ]. Cada elemento tiene una posición (índice) que empieza en 0.

Ejemplo — Catálogo de productos
productos = ["Laptop", "Mouse", "Teclado", "Monitor"]
precios   = [15000, 450, 800, 3200]

# Acceder por posición (empieza en 0)
print(productos[0])   # "Laptop"
print(productos[-1])  # "Monitor" (último)

# Cuántos hay
print(len(productos))  # 4

# Agregar un producto nuevo
productos.append("Webcam")

# ¿Está en la lista?
print("Mouse" in productos)  # True
Operaciones más usadas
OperaciónCódigoQué hace
Acceder lista[0] Obtiene el primer elemento
Último lista[-1] Obtiene el último elemento
Contar len(lista) Número de elementos
Agregar lista.append(x) Agrega al final
Eliminar lista.remove(x) Elimina el valor x
Buscar x in lista True si x existe
Ordenar lista.sort() Ordena la lista
🗂️ Diccionarios REGISTROS

Un diccionario guarda pares de clave: valor entre llaves { }. Es ideal para representar un registro, como una ficha de empleado o un producto.

Ejemplo — Ficha de empleado
empleado = {
    "nombre":       "Ana García",
    "departamento": "Ventas",
    "salario":      18500,
    "activo":       True
}

# Acceder a un valor
print(empleado["nombre"])     # "Ana García"
print(empleado["salario"])    # 18500

# Modificar un valor
empleado["salario"] = 20000

# Agregar una clave nueva
empleado["correo"] = "ana@empresa.com"

# Ver todas las claves
print(empleado.keys())
💡 Diferencia clave: Las listas usan posiciones numéricas [0], los diccionarios usan nombres ["nombre"]. Los diccionarios son mejores cuando los datos tienen etiquetas.
Operaciones más usadas
OperaciónCódigoQué hace
Acceder d["clave"] Obtiene el valor de esa clave
Modificar d["clave"] = val Cambia o agrega un valor
Claves d.keys() Lista de todas las claves
Valores d.values() Lista de todos los valores
Buscar "k" in d True si la clave existe
Eliminar del d["clave"] Elimina esa clave y valor
🔀 Condicionales DECISIONES

Con if / elif / else el programa toma decisiones. Evalúa una condición y ejecuta diferentes bloques de código según si es verdadera (True) o falsa (False). La indentación (4 espacios) es obligatoria.

Ejemplo — Evaluación de desempeño
ventas_mes = 92000
meta       = 100000
porcentaje = (ventas_mes / meta) * 100

if porcentaje >= 100:
    resultado = "¡Meta superada! 🎉"
    bono      = "10%"
elif porcentaje >= 80:
    resultado = "Cerca de la meta 👍"
    bono      = "5%"
else:
    resultado = "Meta no alcanzada 😔"
    bono      = "0%"

print(f"Logro: {porcentaje:.1f}%")
print(f"Estado: {resultado}")
print(f"Bono: {bono}")
Operadores de comparación
OperadorSignificadoEjemplo
==Igual a ventas == meta
!=Diferente de estado != "cerrado"
>Mayor que precio > 1000
<Menor que stock < 5
>=Mayor o igual edad >= 18
<=Menor o igual descuento <= 0.5
⚠️ Atención: Para comparar usa == (dos signos igual). Un solo = es para asignar un valor, no para comparar.
🔄 Ciclos (for) AUTOMATIZACIÓN

El ciclo for repite un bloque de código para cada elemento de una lista. Es ideal para procesar múltiples registros sin repetir código.

Ejemplo — Reporte de ventas semanal
dias   = ["Lun", "Mar", "Mié", "Jue", "Vie"]
ventas = [3200, 4800, 2900, 5100, 4400]

total = 0
for i in range(len(dias)):
    total += ventas[i]
    print(f"  {dias[i]}: ${ventas[i]:,}")

promedio = total / len(dias)
print(f"\nTotal:   ${total:,}")
print(f"Promedio: ${promedio:,.0f}")
Forma simple — for directo sobre lista
productos = ["Laptop", "Mouse", "Teclado"]

for producto in productos:
    print("  ✔", producto)
💡 Cuándo usar cada forma:
• for x in lista — cuando solo necesitas el valor
• for i in range(len(lista)) — cuando necesitas el índice también
⚙️ Funciones REUTILIZACIÓN

Una función es un bloque de código con nombre que puedes usar muchas veces. Se define con def, recibe parámetros de entrada y puede devolver un resultado con return.

Ejemplo — Calculadora de precios
def calcular_precio_final(precio_base, descuento=0):
    # Aplicar descuento
    precio_desc = precio_base * (1 - descuento)
    # Calcular IVA (16%)
    iva   = precio_desc * 0.16
    total = precio_desc + iva
    return round(total, 2)

# Llamar la función
laptop  = calcular_precio_final(12000, descuento=0.10)
mouse   = calcular_precio_final(400)

print(f"Laptop:  ${laptop:,}")
print(f"Mouse:   ${mouse:,}")

Parámetros precio_base, descuentoDatos que recibe la función
Valor por defectodescuento=0 Si no se pasa, usa 0
Cuerpo total = ... El código a ejecutar (indentado)

🖨️ print() y f-strings REFERENCIA

La función print() muestra información en pantalla. Las f-strings permiten insertar variables dentro de un texto de forma elegante.

Formas de usar print()
nombre  = "Tech Store"
ventas  = 85000
margen  = 37.5

# Forma simple con comas
print("Empresa:", nombre)

# f-string (recomendado) — insertar variable con { }
print(f"Empresa: {nombre}")
print(f"Ventas: ${ventas:,}")       # :,  → separador de miles
print(f"Margen: {margen:.1f}%")      # :.1f → 1 decimal
print(f"Ventas: ${ventas:,.0f}")    # :,.0f → miles sin decimales
Formatos útiles en f-strings
FormatoQué haceEjemplo
{x:,} Separador de miles 85,000
{x:.2f} 2 decimales 37.50
{x:.1f} 1 decimal 37.5
{x:,.2f} Miles con 2 decimales 85,000.00
{x:.0f} Sin decimales (redondea) 38
{x:.1%} Porcentaje con 1 decimal 37.5%
